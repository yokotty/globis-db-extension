# GLOBIS マイページ投稿拡張

GLOBIS VC マイページのディスカッション投稿で、  
未いいねの本文を自動で展開し、いいね時には3行表示へ戻す拡張です。  
自分自身の未いいね投稿は、初期表示時に3行表示のまま維持します。

**Features**
- 未いいねの投稿は `more` を自動クリックして全文表示
- 自分自身の未いいね投稿は自動展開せず、3行表示のまま維持
- いいね押下をトリガに、`閉じる` 状態なら自動で3行表示へ戻す
- 投稿本体に同じ挙動を適用
- Lazy load で追加された投稿も自動で再評価
- 既存のネイティブ `more` / `閉じる` ボタンをそのまま利用
- 画面の見やすさ向上のため、ヘッダー/タブ/ツールバーの高さ・余白を自動調整
  - `#tabs`: `md:h-[52px]` -> `md:h-[42px]`
  - ディスカッションツールバー: `md:py-6` -> `md:py-3`
  - `#header-desktop-nav`: `py-[7px]` -> `py-[1px]`

**対象ページ**
- `https://vc.globis.ac.jp/my/cm/*`

**インストール方法（Chrome）**
1. `chrome://extensions` を開く
2. Developer Mode を ON
3. "Load unpacked" をクリック
4. このリポジトリのルートフォルダを選択

**使い方**
- 対象ページを開くと、未いいね投稿が自動で全文表示されます
- いいねを押すと、該当投稿が3行表示へ戻ります
- スクロールで追加された投稿にも自動で適用されます

**更新方法**
1. ファイル変更後、`chrome://extensions` の更新ボタンを押す
2. 対象ページを再読み込み

**テスト**
1. `npm install`
2. `npm test`

**トラブルシューティング**
- 何も変わらない
  - 拡張が有効か確認
  - 対象ページを再読み込み
- 投稿が展開されない
  - 拡張を更新してページを再読み込み
  - 対象URLが `https://vc.globis.ac.jp/my/cm/*` か確認
- 余白調整が反映されない
  - 拡張を更新してページを再読み込み
  - 画面要素のクラス変更が発生していないか確認

**Permissions**
- 追加の権限は不要（content script のみ）
